# **App Name**: VentaLink CRM

## Core Features:

- Detailed Sales Form: A comprehensive form to input sales data including client, product, operator, and sale details.
- Interactive Sales Dashboard: A real-time dashboard visualizing key sales metrics, trends, and performance indicators using charts and graphs.
- Client Management: Dedicated section to add, edit, and manage client information, including contact details and sales history.
- Product Catalog: Functionality to create, update, and categorize products with details like pricing, description, and availability.
- Operator Management: Section to manage sales operators, including assigning roles, tracking performance, and managing contact information.
- Sales Record Database: Database structured around a 'Registro de Venta' object with fields: GRUPO, MES VTA, CARGA, DISTRI, GESTION, SEGMENTO, ASESOR, COORDINACION, COLABORADOR, CAMPAÃ‘A, CLIENTE, DNI, TLF CONTACTO, HISTORICO CLIENTE, OFRECIMIENTO COMERCIAL, PACK, PROMO, TEC, HUELLA, TIPO, SERVICIOS, NUMERO, .OPERADOR, TIPO DE LINEA, ESTADO, SUBESTADO, MES ACT., FCHA ACT., DATOS PORTABILIDAD, DOC ADIC EN CRM, ZONA, FCHA BAJA, DIRECCION, LOCALIDAD, PROVINCIA, COD POSTAL, CUENTA, CORREO, PRECIO CIERRE. .OPERADOR is a dropdown, HISTORICO CLIENTE is multi-line text.

## Style Guidelines:

- Primary color: Moderate blue (#5DADE2) for trust and reliability.
- Background color: Light gray (#E5E8E8), nearly white, to maintain a professional, clean, and neutral backdrop.
- Accent color: Complementary, desaturated blue (#7986CB), adding subtle depth to the design while still resonating professionalism.
- Headline font: 'Poppins', sans-serif. Body font: 'PT Sans', sans-serif. A pairing that gives a precise and clean, yet gently humanist feel.
- Use minimalist, line-style icons for key actions and navigation.
- Implement a clean and intuitive grid layout for easy navigation.
- Subtle transitions and animations to provide feedback on interactions.
